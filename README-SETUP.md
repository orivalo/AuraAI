# Настройка Aura AI

## Настройка Groq API

1. Зарегистрируйтесь на [Groq Console](https://console.groq.com/)
2. Создайте API ключ в разделе API Keys
3. Создайте файл `.env.local` в корне проекта:
   ```
   GROQ_API_KEY=ваш_api_ключ_здесь
   ```
4. Перезапустите сервер разработки:
   ```bash
   npm run dev
   ```

## Использование

После настройки API ключа, приложение готово к использованию. Откройте чат и начните общение с AI-психологом.

## Модель

Используется модель `llama-3.3-70b-versatile` от Groq - быстрая и бесплатная модель для чата.
